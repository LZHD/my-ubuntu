**2. 常见问题**

>* 软件包依赖

```sh
    sudo apt-get update
    sudo apt-get install -f
```
>* 文件管理器卡死

```sh
    pkill nautilus
```
>* 搜狗中文输入法异常

```sh
    rm -rf ~/.config/SogouPY* ~/.config/sogou*
```
